# AI Translator

一款基于浏览器的长篇文档翻译器。支持 TXT、Markdown、HTML 输入，适合长文档分块翻译、术语统一和多平台模型调用。

主要是厌倦了稍微长一点的文字就需要分几次粘贴到翻译对话框；还有就是真这么做的时候，有些名词术语在不同的对话中会被翻译成不同的结果；如果把长文档直接扔AI，受限于上下文又会很不稳定。这个脚本可以解决上述问题。

## 项目说明

这个项目是一个单页网页工具，核心目标是把长文档翻译流程做得更稳一点：

- 支持长文本分块翻译
- 支持术语表抽取与统一
- 支持上下文记忆，减少前后术语漂移
- 支持多家模型平台
- 支持不同翻译风格
- 支持译文和术语表下载

## 功能

- TXT / Markdown / HTML 文件输入
- 多平台 API 支持
  - Mistral
  - xAI
  - OpenAI
  - Google
  - Anthropic
  - 阿里云 Qwen
- 翻译风格
  - 直译模式
  - 忠实自然模式
  - 文学润色模式
- 术语策略
  - 翻译为中文
  - 保留原文
- 实时日志面板
- 译文下载
- 术语词典下载

## 使用方法

1. 打开 `AI Translator.html`
2. 选择要翻译的文件
3. 选择平台和模型
4. 选择翻译风格
5. 选择术语策略
6. 输入 API Key
7. 点击“开始翻译”
8. 翻译完成后，可下载译文和术语表

## 文件命名规则

下载文件会自动按以下格式命名：

- 中文译文：`原文件名 中文译文 模型名 风格名.txt`
- 术语表：`原文件名 术语表 模型名 风格名.csv`

其中风格名使用两个字：

- 直译
- 自然
- 润色

## 支持的输入格式

- `.txt`
- `.md`
- `.html`
- `.htm`

## 注意事项

- 这是一个前端网页工具，需要你自己提供对应平台的 API Key。
- 不同模型的参数支持可能不同，如果某个模型报参数错误，可以切换到更通用的模型。
- 长文本翻译时，建议保持网络稳定，避免中途中断。
