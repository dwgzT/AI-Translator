# AI Translator

一个单 HTML 文件的长篇文档翻译工具。

### 项目初衷

主要是厌倦了稍微长一点的文字就需要分几次粘贴到翻译对话框；还有就是真这么做的时候，有些名词术语在不同的对话中会被翻译成不同的结果；如果把长文档直接扔 AI，受限于上下文又会很不稳定。这个脚本可以解决上述问题。可以非常方便地翻译长篇小说，电影字幕。

### 功能特性

- **自动分块处理**：自动识别段落并切分长文本，每块独立翻译，稳定不截断。
- **SRT 字幕支持**：原生解析 SRT 格式，逐条翻译并保留时间码与特效标签。
- **多目标语言**：支持翻译为中文、英语、德语、西班牙语、日语，自动检测源语言并在与目标语言相同时发出警告。
- **四种翻译风格**：直译模式、忠实自然模式、文学润色模式、学术论文模式。
- **术语一致性**：动态提取并强制遵循术语表，同名词汇全文统一翻译。
- **术语策略**：支持将术语翻译为目标语言，或在正文中保留原文、仅建档译名两种策略。
- **上下文记忆**：翻译时参考最近数段的译文，保持风格与用词连贯。
- **CSV 术语表**：可上传自定义 CSV 术语表，翻译完成后可下载更新后的词典。
- **格式规范化**：导出文本自动清理多余空格与空行，每段间确保单个空行。
- **换行格式选择**：下载时可选 Mac/Linux（LF）或 Windows（CRLF）换行格式。
- **多供应商支持**：Mistral、xAI、OpenAI、Google、Anthropic、阿里云（Qwen）。

### 使用方法

1. 下载 `AI Translator.html` 文件，在浏览器中打开。
2. 选择 AI 供应商、输入 API Key 并选择模型。
3. 上传待翻译文件（TXT / MD / HTML / SRT）。
4. 可选：上传 CSV 术语表以强制统一特定名词的翻译。
5. 选择目标语言、翻译风格及术语策略，点击「开始翻译」。
6. 翻译完成后下载规范化的文本文档或字幕文件，以及更新后的术语词典。

### 术语表格式

CSV 文件第一行为表头，格式为 `原文,目标语言代码`（如 `原文,zh`），之后每行一个术语对：

```
原文,zh
Wolverine,金刚狼
Logan,罗根
Professor X,X教授
```
