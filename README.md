# AI Translator

一个单 HTML 文件的长篇文档翻译工具。

### 项目初衷

主要是厌倦了稍微长一点的文字就需要分几次粘贴到翻译对话框；还有就是真这么做的时候，有些名词术语在不同的对话中会被翻译成不同的结果；如果把长文档直接扔 AI，受限于上下文又会很不稳定。这个脚本可以解决上述问题。

### 功能特性

- 自动分块处理：自动识别段落并切分长文本。
- 术语一致性：动态提取并强制遵循术语表。
- 上下文记忆：翻译时参考前文内容。
- 多翻译模式：支持直译、润色、学术等风格。
- 格式规范化：自动清理空格与空行。
- 多供应商支持：Mistral, OpenAI, Google, Anthropic, 阿里云, xAI。

### 使用方法

1. 下载 AI Translator.html 文件。
2. 在浏览器中打开。
3. 输入 API Key 并上传 TXT, MD 或 HTML 文件。
4. 翻译完成后下载规范化的文本。
